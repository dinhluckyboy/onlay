Onlay.js

Hướng dẫn sử dụng thư viện Onlay.js (Modal) được code hoàn toàn bằng Javascript

1. Lợi ích

Giúp ẩn hiện Modal, có thể tái sử dụng code giúp tiết kiệm thời gian

2. Ví dụ trực quan

Click vào button dưới đây để xem ví dụ chi tiết 👇

3. Cách sử dụng

Liên kết file từ thư viện

Dán đoạn mã này để link file css vào phần header của file HTML 

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/dinhluckyboy/onlay@main/onlay.min.css"/>

Tiếp thep dán đoạn mã này để link file js vào trước thẻ đóng body của file HTML 

<script src="https://cdn.jsdelivr.net/gh/dinhluckyboy/onlay@main/onlay.min.js"></script>

Khởi tạo modal

// Initialize a new modal
const modal = new Onlay({
  content: "<h2>Modal Content</h2>",
  // Use either "content" or "templateId". If both are set, "content" takes precedence.
  templateId: "template-content",
  // Use either "content" or "templateId". If both are set, "content" takes precedence.
  footer: true, // default is false
  destroyOnClose: false,
  closeMethods: ["button", "overlay", "escape"], // default is ["button", "overlay", "escape"]
  cssClass: ["custom-class1", "custom-class2"], // default is []
  onOpen: function () {
    console.log("Modal open");
  },
  onClose: function () {
    console.log("Modal close");
  },
});

// set new content
modal.setContent("<h2>Modal new content</h2>");

// add button footer
modal.addFooterButton("Close", "modal__btn-custom", () => {
  modal.close();
});

// open modal
modal.open();

Khi khởi tạo modal, bắt buộc phải truyền "content" hoặc "templateId", nếu cả 2 được cung cấp, "content" sẽ được ưu tiên, nếu chỉ cung cấp "templateId" hãy đảm bảo rằng truyền đúng "id" trên trang HTML 

<template id="template-content">
      <h2>Template Content</h2>
    </template>

Chú thích về các thuộc tính khởi tạo

Property Type Description
content, templateId string Sử dụng "content" hoặc "templateId". Nếu cả hai đều được thiết lập, "content" sẽ được ưu tiên.
cssClass array Giá trị mặc định là mảng rỗng, có thể truyền vào mảng giá trị chuỗi, giá trị này sẽ là class để bạn tuỳ chỉnh css cho ".onlay__container"
footer boolean Giá trị mặc định là "false", khi bạn truyền "true", modal sẽ cho phép add button vào footer hoặc set lại content cho footer bằng method addButtonFooter(), setFooterContent()
destroyOnClose boolean Giá trị mặc định là "false", khi truyền là "true" khi đóng modal HTML DOM sẽ không bị xoá khi đóng modal và ngược lại
closeMethods array Giá trị mặc định là mảng ["button", "overlay", "escape"], "button" tương ứng với cho phép đóng modal bằng nút button close, "overlay" cho phép đóng modal bằng cách click vào lớp phủ overlay, "escape" cho phép đóng modal bằng cách nhấn phím Esc trên bàn phím, nếu xoá các giá trị này modal sẽ không cho phép đóng tương ứng với giá trị đã xoá
onOpen function Hàm sẽ đươc gọi sau khi modal được mở
onClose function Hàm sẽ được gọi sau khi modal đóng

4. API

Method Description
open() Dùng để mở modal
close(boolean) Dùng để đóng modal, có thể không truyền tham số, khi không truyền thì giá trị mặc định là "false", khi là "fasle" lúc đóng modal code sẽ được xoá khỏi DOM và ngược lại
destroy() Dùng để đóng modal và code sẽ được xoá khỏi DOM
setContent(content) Dùng để set lại content cho modal, typeof content === "string"
setFooterContent(content) Dùng để set lại content cho footer, typeof content === "string"
addFooterButton(title, cssClass, callback) Dùng để add button cho footer, title = string là tiêu đề của button, cssClas là mảng chứa các class để custom cho button, callback là hàm được gọi sau khi onclick button
